GitHub / Cloud Setup
How to deploy this app and access your data anywhere in the world.
Important
This HTML file alone cannot magically sync data worldwide. To access your records and users anywhere in the world, you need: 1. A hosted website, such as GitHub Pages 2. A cloud database, such as Firebase Firestore
GitHub Pages Steps
1. Create a GitHub account 2. Create a new repository 3. Upload this HTML file as index.html 4. Open repository Settings → Pages 5. Set source to main branch / root 6. Your site will be live on a public GitHub Pages URL
Cloud Database Steps
1. Create a Firebase project 2. Enable Firestore Database 3. Replace the FIREBASE_CONFIG object in this file with your project config 4. Set USE_CLOUD = true in the script 5. Upload again to GitHub Pages 6. Your users and records will sync online
